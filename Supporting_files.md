Adventureliib
-------------
The library adventurelib can be downloaded from here: https://pypi.org/project/adventurelib/#files.
Alternitavely, you can go to PyPI yourself and search the library and download it there. 
After doing this, you can pip install the library using "pip install adventurelib" in the terminal.

Random and time
---------------
Because random and time are both built in libraries, they are installed in the code.
